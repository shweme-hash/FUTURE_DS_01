Excel dashboard analyzing Superstore sales data to uncover key insights on revenue, regions, and top-performing customers.
ashboard)

📌 Project Overview

This project analyzes retail sales data to uncover key business insights such as revenue trends, top-performing products, and regional performance.

The goal is to simulate a real-world data analyst task by transforming raw sales data into a meaningful and interactive Excel dashboard that supports business decision-making.

🎯 Objectives
- Identify top-performing products and customers
- Analyze sales distribution across regions and categories
- Track overall business performance using key metrics
- Provide insights and recommendations for growth

🛠️ Tools Used

- Microsoft Excel (Data Cleaning, Analysis, Dashboard Creation

📁 Dataset

[Superstore Dataset (Kaggle)](https://www.kaggle.com/datasets/vivek468/superstore-dataset-final)

 Data Cleaning Process

- Removed 6 rows with missing values in:
  - Sales
  - Profit
  - Discount
- Ensured correct data types (numeric and date formats)
- Verified consistency across key columns

📊 Dashboard Features

The Excel dashboard includes the following key metrics and visualizations:

🔢 Key Performance Indicators (KPIs)

- Total Sales
- Total Orders
- Average Sales

📈 Visual Insights

- Sales by Category
- Sales by Region
- Top 10 Cities by Sales
- Top 5 Customers by Sales

🔍 Key Insights

- Certain categories contribute significantly more to total sales
- A few cities dominate overall revenue generation
- Sales are concentrated among top customers (high-value clients)
- Regional performance varies, indicating potential growth opportunities


💡 Recommendations

- Focus marketing efforts on high-performing categories
- Expand strategies in top-performing regions
- Build retention strategies for top customers
- Investigate underperforming regions for improvement opportunities
